# Prompt para Criar uma Rota de API

Crie a rota `PATCH /workout-plans/{id(uuid)}/days/{id(uuid)}/sessions/:id(uuid)`

## Descrição
Ela atualiza uma sessão de treino específica. Marca como concluída, atualizando WorkoutSession.completedAt

## Autenticação

- Rota protegida.
- O usuário PRECISA ser dono do plano de treino recebido

    completedAt: string; //ISO STRING DATE '2026-03-03T20:22:56.246Z'
}
