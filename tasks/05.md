# Prompt para Criar uma Rota de API

Crie a rota `GET /workout-plans/:id/days/:id`

## Descrição

Essa rota deve retornar o dia de treino recebido, seus exercícios, e para cada exercício, suas sessões (workout sessions).

## Requisitos Técnicos

N/A

## Autenticação

- Rota protegida.
- O usuário precisa ser dono do plano de treino recebido.

## Request

```ts
interface Body {}
```

```ts
interface Params {
  date: string;
}
```

```ts
interface Query {}
```

## Response

```ts
interface StatusCode200 {
  id: string;
  name: string;
  isRest: boolean;
  imageCoverUrl?: string;
  estimatedDurationInSeconds: number;
  exercisesCount: number;
  exercises: Array<{
    // Todos os campos do model WorkoutExercise, com excessão de createdAt e updatedAt
    id: string;
    name: string;
    order: number;
    sets: number;
    reps: number;
    restTimeInSeconds: number;
    workoutDayId: string;
  }>;
  weekDay: string;
  sessions: Array<{
    workoutDayId: string;
    startedAt?: string; //YYYY-MM-DD
    completedAt?: string; //YYYY-MM-DD
  }>
}
```
